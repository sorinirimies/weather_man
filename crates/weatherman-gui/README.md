# weatherman (GUI)

Desktop weather app built with [Iced](https://iced.rs/). Shows current conditions,
the next 24 hours, and a 7-day forecast for any city (or your auto-detected location).

Powered by [`weatherman-core`](https://crates.io/crates/weatherman-core) —
data from [Open-Meteo](https://open-meteo.com/), no API key required.

## Install

```
cargo install weatherman
```

## Run

```
weatherman
```

Type a city name and press Enter to search, or toggle °C/°F with the units button.
